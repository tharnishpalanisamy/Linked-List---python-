Linked List in Python

This repository contains a complete implementation of the Linked List data structure in Python. It is built entirely from scratch without relying on Pythonâ€™s built-in list, allowing a clearer understanding of how Linked Lists work internally. The project covers all fundamental operations such as insertion, deletion, traversal, searching, updating, and reversing. It also includes implementations of both singly and specialized linked list operations where applicable.

In addition to the core data structure, the repository includes solutions to all the Linked List problems I have solved on LeetCode. These solutions demonstrate how theoretical Linked List concepts are applied to real coding interview challenges.

Features

Custom Node class and pointer-based structure

Insert operations (beginning, end, specific index)

Reverse the linked list

Calculate the length of the list

Full traversal and display methods
